## Introduction :bulb:
- **혼잡 여부 예측 모델**은 코로나 19 등 혼잡하지 않은 약속 장소를 원하는 서울권 거주자를 위한 모델입니다.
- 교통, 상권, 날씨 등을 고려하여, 혼잡할 확률을 제시하고 해당 인근 지역의 관광지를 제시하는 Toy 서비스를 개발했습니다.

## 1. 프로젝트 소개
### 1-1. 배경
  - 지인과의 만남을 계획할 때 약속 장소를 정하는 일이 쉽지 않고, 혼잡하지 않은 장소를 선호한다.
  - 최근 코로나 19 이슈로 인하여, 혼잡하지 않은 장소에 대한 니즈가 증가했다.
  - 약속 장소로 고려 중인 지역의 혼잡 여부를 예측할 수 있다면, 위와 같은 고민이 줄어들 것이다.
### 1-2. 프로젝트 개요
1. 프로젝트명 : 교통, 상권, 날씨를 고려한 혼잡 여부 예측 모델 개발
2. 수행자 : 플레이스투밋(P2M) - 이민호, 정재영, 이서영
3. 수행기간 : 2주 (2021. 12월 중)
4. 목표 : 지하철역 기준 인근 지역의 혼잡 여부 및 해당 확률 제시
5. 데이터셋 - 공공 데이터 활용
